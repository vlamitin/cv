# CV

## What is it
CV, created with [this template](https://github.com/EyalPerry/dev-cv)

## Prerequisites:
- nodejs v 12.3.*
- npm v 6.*

## Installation
- `npm ci`

## Usage
- `npm run start`
- edit https://github.com/vlamitin/cv/blob/master/public/cv.json
- save pdf with chrome browser
- or run `make pdf` (currently only linux supported)
